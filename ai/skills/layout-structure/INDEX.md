# Layout & Structure Skills

Nhóm này gom các skill liên quan đến bố cục trang, lưới, section, scroll behavior và cấu trúc layout tổng thể.

```
ai/skills/layout-structure/
|-- bento-grid/            -> bento layout, feature blocks và asymmetric grid
|-- footer/                -> footer, sitemap, legal links và bottom navigation
|-- grid-system/           -> grid system, columns, gutters và responsive layout
|-- hero-section/          -> hero section, first viewport và visual signal
|-- infinite-scroll/       -> infinite scroll, sentinel, load more và pagination fallback
|-- parallax/              -> parallax section, depth motion và scroll-linked visual
|-- scroll-snap/           -> scroll snap sections, carousels và paged scroll
|-- split-layout/          -> split screen, two-column layout và side-by-side content
`-- sticky-section/        -> sticky section, pinned content và scroll storytelling
```

Mỗi skill có `SKILL.md` riêng trong folder con tương ứng.
