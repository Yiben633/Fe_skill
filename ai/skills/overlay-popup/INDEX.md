# Overlay & Popup Skills

Nhóm này gom các skill liên quan đến lớp nổi, popup, menu ngữ cảnh và thông báo tạm thời.

```
ai/skills/overlay-popup/
|-- alert-banner/          -> alert, banner và inline/system notice
|-- bottom-sheet/          -> bottom sheet mobile và action sheet
|-- command-palette/       -> command palette, search command và action launcher
|-- context-menu/          -> menu chuột phải, contextual actions
|-- drawer/                -> side drawer, panel và slide-over
|-- dropdown/              -> dropdown menu, select-like menu và menu button
|-- modal/                 -> modal, dialog và confirmation flow
|-- popover/               -> popover giàu nội dung và anchored panel
|-- toast-snackbar/        -> toast, snackbar và transient feedback
`-- tooltip/               -> tooltip ngắn, label phụ và affordance hint
```

Mỗi skill có `SKILL.md` riêng trong folder con tương ứng.
