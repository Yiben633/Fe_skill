# Navigation Skills

Nhóm này gom tất cả skill liên quan đến điều hướng trong app, website và shell UI.

```
ai/skills/navigation/
|-- breadcrumb/            -> đường dẫn phân cấp và điều hướng ngữ cảnh
|-- mega-menu/             -> menu lớn nhiều cột, nhóm link và preview
|-- mobile-hamburger/      -> menu mobile, drawer, overlay và focus handling
|-- navbar/                -> sticky, transparent blur, hamburger mobile
|-- pagination/            -> phân trang, page size và điều hướng dữ liệu
|-- sidebar/               -> sidebar navigation, collapse và active state
|-- stepper/               -> tiến trình nhiều bước, wizard và checkout flow
`-- tab-bar/               -> tabs, segmented nav và active tab behavior
```

Mỗi skill có `SKILL.md` riêng trong folder con tương ứng.
