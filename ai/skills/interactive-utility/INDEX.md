# Interactive Utility Skills

Nhóm này gom các skill tiện ích tương tác thường dùng trong app, dashboard, editor và tool UI.

```
ai/skills/interactive-utility/
|-- copy-to-clipboard/     -> copy action, feedback và clipboard fallback
|-- dark-mode-toggle/      -> theme switch, system preference và persistence
|-- drag-drop/             -> drag and drop, reorder, dropzone và sortable UI
|-- filter-sort-bar/       -> filter bar, sort controls, chips và query state
|-- infinite-canvas/       -> canvas vô hạn, pan/zoom, nodes và viewport
|-- resizable-panel/       -> resizable panes, splitters và layout persistence
|-- search-ui/             -> search box, results, suggestions và empty state
|-- share-sheet/           -> share sheet, native share và social/share actions
`-- virtual-scroll/        -> virtualized list/grid, windowing và large data UI
```

Mỗi skill có `SKILL.md` riêng trong folder con tương ứng.
