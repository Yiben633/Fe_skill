# Animation Parents

Nhóm này gom các skill liên quan đến animation patterns, micro-interactions, scroll effects và motion-rich UI.

```
ai/skills/animation-parents/
|-- counter-animation/     -> count-up number, metric reveal và numeric motion
|-- cursor-custom/         -> custom cursor, pointer trail và interactive cursor
|-- lottie-svg-anim/       -> Lottie, SVG animation và animated illustration
|-- magnetic-button/       -> magnetic hover, pointer attraction và CTA motion
|-- morphing-shape/        -> shape morphing, blob/path transition và SVG morph
|-- page-transition/       -> route/page transition và view enter/exit
|-- particle-system/       -> particles, canvas background và interactive field
|-- scroll-animation/      -> skill tổng quan cho scroll animation patterns
|-- scroll-progress/       -> scroll progress bar, reading progress và section progress
|-- scroll-reveal/         -> reveal theo viewport: fade, slide, stagger
|-- stagger-entrance/      -> staggered entrance cho list, grid và section children
`-- text-split-animate/    -> split text, line/word/char reveal và kinetic type
```

Mỗi skill có `SKILL.md` riêng trong folder con tương ứng.
