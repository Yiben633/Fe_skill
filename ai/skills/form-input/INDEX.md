# Form & Input Skills

Nhóm này gom các skill liên quan đến nhập liệu, lựa chọn, picker và form controls.

```
ai/skills/form-input/
|-- checkbox-radio/        -> checkbox, radio group và selection controls
|-- color-picker/          -> chọn màu, swatch, palette và alpha control
|-- date-picker/           -> date picker, range picker và calendar popover
|-- file-upload/           -> upload file, drag-drop, preview và progress
|-- input-field/           -> text input, label, helper, error và validation
|-- otp-input/             -> OTP, PIN, verification code input
|-- rating/                -> star rating, score picker và feedback rating
|-- rich-text-editor/      -> editor, toolbar, formatting và content states
|-- select-combobox/       -> select, combobox, autocomplete và async options
|-- slider-range/          -> slider, range slider và numeric controls
|-- tag-input/             -> tag entry, token input và creatable chips
|-- textarea/              -> multiline input, auto-resize và character count
`-- toggle-switch/         -> toggle, switch và binary setting
```

Mỗi skill có `SKILL.md` riêng trong folder con tương ứng.
