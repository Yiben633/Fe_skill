# Feedback & Status Skills

Nhóm này gom các skill liên quan đến phản hồi trạng thái, loading, lỗi, dữ liệu trống và tín hiệu định danh nhỏ.

```
ai/skills/feedback-status/
|-- avatar/                -> avatar, initials, presence và fallback identity
|-- badge-chip/            -> badge, chip, tag, pill và status label
|-- empty-state/           -> trạng thái trống, no results và onboarding empty
|-- error-state/           -> lỗi, retry, unavailable và degraded state
|-- progress-bar/          -> progress bar tuyến tính và completion state
|-- skeleton-loader/       -> skeleton loading, shimmer và placeholder layout
|-- spinner-progress/      -> spinner, indeterminate progress và loading affordance
`-- status-dot/            -> status dot, presence, health và live indicator
```

Mỗi skill có `SKILL.md` riêng trong folder con tương ứng.
