# Content Display Skills

Nhóm này gom các skill dùng để hiển thị, sắp xếp và trình bày nội dung trong UI.

```
ai/skills/content-display/
|-- accordion/             -> nội dung thu gọn/mở rộng và FAQ
|-- card-grid/             -> grid card, masonry nhẹ, responsive collections
|-- carousel/              -> carousel item, controls, autoplay và snapping
|-- data-grid/             -> data grid nhiều cột, sort, filter, selection
|-- feed/                  -> activity feed, social feed và timeline stream
|-- gallery/               -> gallery ảnh/video, thumbnail và lightbox
|-- list/                  -> list item, dense list, selectable list
|-- marquee/               -> marquee nội dung chạy ngang có kiểm soát
|-- slideshow/             -> slideshow, deck và slide transitions
|-- table/                 -> table cơ bản, responsive table và row states
|-- ticker/                -> ticker tin tức, giá, trạng thái realtime
`-- timeline/              -> timeline dọc/ngang, milestone và events
```

Mỗi skill có `SKILL.md` riêng trong folder con tương ứng.
