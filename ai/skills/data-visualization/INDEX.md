# Data Visualization Skills

Nhóm này gom các skill liên quan đến trực quan hóa dữ liệu, metric, chart và tín hiệu số.

```
ai/skills/data-visualization/
|-- bar-chart/             -> bar chart, grouped/stacked bars và ranking
|-- gauge-dial/            -> gauge, dial, meter và threshold indicator
|-- heatmap/               -> heatmap, calendar heatmap và intensity matrix
|-- line-chart/            -> line chart, trend, time series và multi-series
|-- pie-donut/             -> pie chart, donut chart và part-to-whole data
|-- sparkline/             -> sparkline, mini trend và inline chart
|-- stat-card/             -> KPI card, metric card, trend và dashboard stats
`-- tree-map/              -> treemap, hierarchical area và nested proportions
```

Mỗi skill có `SKILL.md` riêng trong folder con tương ứng.
